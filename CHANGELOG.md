# Changelog

## [2.0.0] - 2026-06-11

### 重大变更
- 🎯 **Qt6 迁移** — 从 MFC 迁移到 Qt6 Widgets 框架，支持跨平台

### 新增特性
- 全新的 Qt6 图形界面
- 支持加载和编辑 Diablo II 角色存档 (.d2s)
- 角色基础信息查看和编辑
- 装备/物品查看和编辑
- 技能点编辑
- 任务信息查看
- 传送点编辑
- 铁匠铺铸造系统
- 物品库查询

### 技术改进
- 使用 CMake 构建系统替换 Visual Studio 项目
- 支持 Linux 编译运行
- 代码结构优化，清晰的模块分层（core / data / ui）
- 强大的物品数据解析引擎
